## 产品简介

**龙虾管家**是 [OpenClaw](https://github.com/openclaw/openclaw) AI Agent 运行时的桌面图形客户端，基于 [ClawX](https://github.com/ValueCell-ai/ClawX) v0.4.15 深度品牌定制。

提供可视化界面来管理、交互和监控 AI Agent，支持 **Windows / macOS / Linux** 三平台。

### 核心特性

- 🤖 **多 Agent 管理**：创建、配置、切换多个 AI Agent
- 🔌 **插件生态**：支持企业微信、钉钉、微信服务号等平台插件
- 🛠️ **技能市场**：安装和管理 OpenClaw Skills
- 📊 **用量统计**：Dashboard 展示 Token 使用趋势
- 🌐 **本地化**：完整中文界面 + 聚星逸 AI Provider 预配置
- 💾 **本地存储**：无需数据库，配置加密存储于本地

### 快速开始

```bash
git clone git@github.com:zhaobod1/openclaw-butler-jarvis.git
cd openclaw-butler-jarvis
corepack enable && corepack prepare
pnpm run init
pnpm dev
```
