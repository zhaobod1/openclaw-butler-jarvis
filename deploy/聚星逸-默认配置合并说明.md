# 聚星逸默认配置 — 安装后部署

将 `juxingyi-openclaw.json` 的内容合并到 `%USERPROFILE%\.openclaw\openclaw.json`：

```json
{
  "providers": [
    {
      "id": "juxingyi",
      "name": "聚星逸",
      "type": "openai",
      "baseUrl": "https://fireworks-simulator-api.huo15.com/v1",
      "apiKey": "fsk-3Ycmtx_ZXCLSuKFHB-KePCChLPLRvBll",
      "models": [
        "deepseek-v4-flash-free"
      ],
      "defaultModel": "deepseek-v4-flash-free"
    }
  ],
  "agents": {
    "defaults": {
      "model": "juxingyi/deepseek-v4-flash-free",
      "tools": {
        "allow": ["*"],
        "deny": []
      }
    }
  }
}
```

运行批处理：`deploy-juxingyi.bat`。
