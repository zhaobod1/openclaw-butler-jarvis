# 龙虾管家 USB 部署包

青岛火一五信息科技有限公司

## 使用方法

1. 将 CLAWBUTLER U 盘插入 Windows 电脑
2. 运行 `build-on-windows.bat` 构建安装包（需要 Node.js + pnpm）
3. 运行生成的 `release/龙虾管家-*-x64-setup.exe` 安装

或直接复制 `deploy-juxingyi.bat` + `juxingyi-openclaw.json` 到已有安装环境。

## 包含

- 龙虾管家 (fork of ClawX) 完整源码
- 聚星逸 AI provider 默认配置（含 API key）
